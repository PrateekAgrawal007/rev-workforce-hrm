# Rev Workforce – HRM System

## Description
Console-based Human Resource Management System built using Java, JDBC, and MySQL.

## Features
- Role-based Login (Employee, Manager, Admin)
- Leave Management & Approval
- Notification System
- Performance Management
- Admin Control Panel

## Technologies
- Java
- JDBC
- MySQL
- IntelliJ IDEA

## Architecture
Layered Architecture (Console → DAO → Database)
